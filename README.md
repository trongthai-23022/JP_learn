# back_end
Cài đặt:
`composer install
`
`cp .env.example .env
`
Cập nhật dữ liệu:
`DB_CONNECTION=mysql          
DB_HOST=127.0.0.1            
DB_PORT=3306                 
DB_DATABASE=tendatabase      
DB_USERNAME=root             
DB_PASSWORD= `
Tạo key:
`php artisan key:generate`
Tạo data:
`php artisan migrate
php artisan db:seed
`
Khởi động:
`php artisan serve`
# font_end
Cài đặt:
`npm install`
Khởi động:
`npm start`
# ai_server
